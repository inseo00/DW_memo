서버 :  서비스를 제공하는 주체
HTML - 웹 서버 - 디비 서버(My SQL)
*HTML에서 바로 데이터베이스로 저장 불가능
id : 변수이름 (중복 X)
name : 변수 이름 (중복 O)
value : 서버에 전송할 실제 데이터
        or  디폴트 값

input type은 한가지 종류만 올 수 있고, 속성(id, name, readonly)은 중복이 가능하다.

DML : 데이터 조작 언어

컬럼 이름 대문자로

1. SELECT
2. DELETE
3. UPDATE
4. INSERT

SELECT 컬럼들, 
FROM 테이블 이름
WHERE 필터링 조건
(>, < =, like)
GROUP BY : 데이터를 그룹화
HAVING : 그룹된 데이터에서 집계 함수 (sum, agv, min, max)
ORDER BY : ASC(오름차순 디폴트 값),
               DESC(내림차순)

LIMIT 데이터 row 강제로 자를 때
테이블 : 실제 데이터를 저장하는 공간
데이터베이스 :  실제 테이블을 저장하는 공간
데이터베이스 - 테이블 - 데이터

컬럼이름이 name에 null이 들어간 데이터 조회할 때 is null 사용
